# Changelog

このファイルは、GitHub のタグやリリースに対応する変更概要をまとめるための記録です。

## v1.0.0 - 2026-03-20

初回の安定リリースです。ローカルで遊べる繁体字単語トレーニングゲームとして、ゲーム進行、語彙生成、外部辞書導線、主要テスト基盤を一通り揃えました。

### Added

- Level 1-3 の繁体字単語を、日本語4択で学べる 1 ページ構成のゲーム UI
- ピンイン表示、カタカナ補助、ブラウザ標準の音声読み上げ
- 開始パネル、レベル別の最高 `Score / Streak` 表示、`localStorage` 保存
- 3回連続ミスで終了するルールと、連続正解ボーナスを含むスコア設計
- Google 翻訳と Weblio への外部確認リンク
- `TOCFL + MJdic + data/manual-vocabulary.json` を使った語彙生成フロー
- Vitest / Playwright / coverage を含む主要テスト基盤

### Changed

- 日本語 UI 文言を全体的に自然な表現へ調整
- ゲームオーバー後の即リトライと開始画面復帰を両立
- 開始前後でレイアウトが跳ねにくいように、主要パネルの高さと導線を調整
- 手修正語彙を拡充し、台湾で使いやすい重要語を増強

### Fixed

- レベル切替や初期化失敗時に古い状態や古いエラーが UI を壊す問題
- 音声未対応環境や `voiceschanged` 発火時の誤案内と二重再生
- 誤答候補の重複、日本語訳の説明調混入、MJdic の空発音欄による生成失敗
- 母音始まり音節のカタカナ補助と、開始画面 / ゲームオーバー時の回復可能エラー表示
